# MyZhihu
这是一个知乎客户端，使用了知乎的http接口
使用了material design设计语言
运用了MVP架构实现，网络请求部分使用了当前流行的retrofir + RXjava + okhttp,使用GSON解析数据，glide获取图片，butterknife自动生成view注解等
项目做了较多的注释，非常适合初学者学习运用
